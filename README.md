# AI Agent Study

강의에서 다룰 5-6가지 프레임워크를 주제별 폴더로 정리하는 레포입니다.

## 주제별 실습
- 예제 에이전트 실습 (`example-agent/README.md`)

## 구조
- 각 주제(프레임워크)마다 루트에 폴더를 하나씩 추가합니다.
- 각 폴더에는 해당 주제의 목적, 실습 내용, 실행 방법을 적은 `README.md`가 있습니다.

## 새 실습 추가 방법
1. 루트에 새 주제 폴더 생성 (예: `framework-foo/`)
2. 폴더 안에 `README.md` 작성
3. 이 루트 `README.md`에 링크 추가
